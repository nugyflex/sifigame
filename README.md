## Sifi Game

## Instructions

| Key | Description          |
| ------------- | ----------- |
| Movement |
| ------------- | ----------- |
| 'w' | Go Forward |
| 'a' | Go Left |
| 's' | Go Backward |
| 'd' | Go Right |
| 'a' | Go Left |
| ------------- | ----------- |
| Barriers |
| ------------- | ----------- |
| 'x' | Place a Vertical Barrier |
| 'z' | Place a Horizontal Barrier |
| ------------- | ----------- |
| Weaponry |
| ------------- | ----------- |
| Left/Right Click | Fire Red Laser |
| Spacebar | Fire Green Laser |
| ------------- | ----------- |
| Miscellaneous |
| ------------- | ----------- |
| 'k' | Enter Debug Mode |
| 'e' | Disable Enemy AI |
| 'f' | Begin New Round |
| ------------- | ----------- |
